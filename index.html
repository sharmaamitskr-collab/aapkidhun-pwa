❓ Help & Guide

AapkiDhun ka poora tutorial

🎵 Prompt Builder

Genre, Language, Vocal, Instruments choose karo → Generate karo → Suno/Udio me paste karo. Limit: 900 chars max. No external links output me.

🔬 Analyze / 🎼 Transcribe

MP3/WAV/MP4 phone se upload karo (📂 Choose File button dabao) ya YouTube/SoundCloud link dalo → Music Sheet milega: MIDI notes, Chords, Beat map, Vocal notes, ABC notation.

✍️ Lyrics Studio

3 tabs: 🆕 New = fresh lyrics banao | 📝 Original = apni lyrics analyze karo | 🔗 Reference = kisi song ke style me nayi lyrics banao. 14+ languages, 22+ vocal types, Auto Provider option.

🌿 Nature Sounds

Rain, Thunder, Birds, Wind, Forest, Animals — chips tap karo select karne ke liye, intensity aur layer choose karo, Generate Prompt karo.

🎧 Offline Player

📂 Load Music dabao → phone ki MP3/WAV files select karo → Prev/Play/Next se chalao. Bina internet ke bhi kaam karega.

📶 PWA Install

Browser me "Add to Home Screen" option se install karo. Ek baar install hone ke baad app bina internet ke bhi poori tarah kaam karega (offline mode).

🎯 Suno Best Practices

  • Prompt ≤ 900 characters rakho
  • Genre + BPM + Key pehle likho
  • Vocal type clearly specify karo
  • No external links ya URLs
  • Nature sounds ke liye intensity mention karo
  • "Reinforcement" line last me zaroor daalo
  • Style Pack me regional naam likho (e.g., Rajasthani-Marwadi)
